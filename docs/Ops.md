# Learn4x Ops

## Arithmetic Ops

### Element-wise and Broadcasting

Add: Z = X + Y

Subtract: Z = X - Y

Multiply: Z = X * Y

Divide: Z = X / Y

### Special

MatMul: Z = X @ Y


## Transform Ops

### Element-wise and independent

Absolute: Z = abs(X)

Exponential: Z = exp(X)

Sine: Z = sin(X)

Cosine: Z = cos(X)

Step: Z = X > 0 ? 1 : 0 // Step Function

Sign: Z = X > 0 ? 1 : (X < 0 ? -1 : 0)

Reciprocal: Z = 1/X

Round: Z = Math.round(X)

Square: Z = X * X

SquareRoot: Z = X ^ 0.5

RSqrt: Z = 1 / X ^ 0.5
